# Patent Disclosure — sourcetrace

**Inventor:** Haley Ann Bird | **Priority Date:** June 3, 2026

## Claims

### Claim 1 — Automated Source Truth Extraction
A system for automatically parsing codebases and extracting verified behavioral guarantees, API contracts, and system invariants.

### Claim 2 — Documentation Divergence Detection
A method for comparing extracted source truth against existing documentation to identify inaccuracies and outdated claims.

### Claim 3 — Structured Reality Map Generation
A system that produces machine-readable and human-readable reality maps from arbitrary codebases, capturing what the system actually does vs. what is claimed.
